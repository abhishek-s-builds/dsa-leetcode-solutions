# Reverse String

## 🧩 Problem
Given a string, reverse the string in-place.

## 🔗 LeetCode Link
https://leetcode.com/problems/reverse-string/

## 💡 Approach (Two Pointers)
- Use two pointers:
  - One at start (left)
  - One at end (right)
- Swap characters
- Move left++ and right--

## ⏱ Time Complexity
O(n)

## 🧠 Space Complexity
O(1)
