# Two Pointers Approach

## 📌 Overview

This folder contains problems solved using the Two Pointers technique.

Two Pointers is an efficient approach used to reduce time complexity, especially in problems involving arrays and strings.

---

## 🚀 When to Use Two Pointers?

* Sorted arrays
* Reversing arrays/strings
* Finding pairs with a given sum
* Removing duplicates
* Sliding window type problems

---

## ⚡ Common Patterns

* Left & Right pointer (start and end)
* Fast & Slow pointer
* Opposite direction traversal
* Same direction traversal

---

## 🧠 Time Complexity Advantage

Most problems reduce from **O(n²) → O(n)** using this approach.
